# Change Log

### v1.1.0

Position parameter now takes a constant instead of an integer, following this convention:

* 1: `top-left`
* 2: `top-center`
* 3: `top-right`
* 4: `center-left`
* 5: `center-center`
* 6: `center-right`
* 7: `bottom-left`
* 8: `bottom-center`
* 9: `bottom-right`
